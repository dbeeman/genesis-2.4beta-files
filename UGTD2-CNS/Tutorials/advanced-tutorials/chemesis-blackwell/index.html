Modeling Calcium Concentration and Biochemical Reactions

by Avrama Blackwell

This directory contains the files from:

Blackwell KT (2005) Modeling Calcium Concentration and Biochemical Reactions. Brains, Minds, and Media 1:bmm224 (urn:nbn:de:0009-3-2245). [Viewable at http://www.brains-minds-media.org/archive/224]

It also contains slide presentations and examples from the Second Latin American School on Computational Neuronscience (LASCON), held in Ribeirao Preto, Brazil in July 2008.

The files included here are:

  • chemesis2.1.tar.gz - chemesis Ver. 2.1

    This is a library of biochemical reaction objects for modeling calcium concentration, contributed by K. T. (Avrama) Blackwell These were used in the paper:

    K.T. Blackwell (2000) Evidence for a Distinct Light-Induced Calcium-Dependent Potassium Current in Hermissenda Crassicornis, J. Computational Neuroscience, 9: 149-170.

    The latest version is available from

    http://www.gmu.edu/departments/krasnow/CENlab/chemesis.html

  • Blackwell_bmm224_2005.pdf - The tutorial as published in Brains, Minds, and Media
  • blackwell-examples.tar.gz - Example scripts to accompany the tutorial
  • blackwell-examples-2008.tar.gz - Updated example scripts to accompany the tutorials given at LASCON 2008.
  • README_2008 - describes the examples in blackwell-examples-2008.
  • Lascon2biochemTutorial.ppt - Slides for LASCON 2008 "Modelling Biochemical Reactions"
  • Lascon2calciumTutorial.ppt Slides for LASCON 2008 "Modelling Calcium Concentration"
