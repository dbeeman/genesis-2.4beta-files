Messages from and to external objects

There are two methods to deal with the fields that hsolve computes. The first one is via regular Genesis messages. These messages must be created on the modeling elements before the SETUP call. During byte-code compilation, hsolve will examine all encountered elements for incoming and outgoing messages and remember to handle these messages during simulation time. If hsolve cannot handle such a message, it will give a warning message in the terminal.
