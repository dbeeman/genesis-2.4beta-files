Parallel GENESIS User Guide

This User Guide is intended for parallel GENESIS users in a workstation environment. If you are using the Cray T3D, please consult the architecture-specific T3D User Guide.

  • User Setup - what steps each user should do to get going.
  • Running Examples - how to run the example parallel scripts.
  • Writing Parallel Scripts - what new commands are in the parallel script language and how to use them.
  • Debugging Parallel Scripts - how to get your parallel scripts to run correctly.
